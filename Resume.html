Materi pembelajaran hari ini

  • Apa itu Git

    • Git Website/Tools untuk remote save, dengan record setiap perubahan. Bisa juga digunakan untuk joint project

  • Cara membuat repository

    • Membuat Account dan repository di GitHub

  • Push File ke Git

    • Push file bisa dilakukan dengan tahapan

    • belakukan inisiasi dengan "git innit"
    • melakukan save file dan kemudian commit dengan command "git commit -m 'perubahan yang dilakukan' "
    • apabila repository baru, buat branch master dengan git branch -m "nama branch"
    • tentukan alamat repository dengan "git remote add origin "alamat.html"
    • gunakan "git push" untuk mengirim file ke git hub

    Note: tambahkan "origin" saat penambahan alamat dan push pertama kali

  • Cloning File
    • cloning file bisa dilakuukan dengan cara memperoleh izin kontribusi dari pemilik file
    • memperoleh alamat file target dari repository pemilik file
    • gunakan command "git clone 'link file.html' untuk melakukan cloning

    Note: pastikan folder cloning tepat

  • Joint Project dan Creating Branch

    • Dalam joint project terdapat command command yang bisa digunakan dalam navigasi branch di joint project

    • git branch -a : Untuk menampilkan list branch
    • git checkout "nama branch" : untuk pindah ke branch target
    • git fetch : Check joint project dengan kondisi terkini
    • git branch -b : membuat branch baru
    • git pull : Update file kita dengan yang terkini
